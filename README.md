> A modern Windows desktop shell inspired by Caelestia Shell.

> 🚧 Work in Progress

## Vision

Wice is a desktop shell and ricing framework for Windows built with C# and Avalonia UI.

The goal is to recreate the polished experience of Caelestia Shell on Windows while embracing native Windows APIs and customization.

## Planned Features

- 🎨 Modern Top Bar
- 📦 Widget Framework
- 🚀 App Launcher
- 🔔 Notification Center
- 🎛 Control Center
- 📊 Dashboard
- 🖥 GlazeWM Integration
- 🎵 Media Controls
- ⚡ Fast Startup
- 🪶 Lightweight
- 🎭 Theme Engine
- 🔌 Plugin System

## Technology Stack

- C#
- .NET 10
- Avalonia UI
- MVVM
- Win32 APIs
- SQLite (planned)
- YAML Configuration

## Project Status

Currently under active development.

## Goal

Create a polished Windows desktop shell inspired by Caelestia Shell that feels native to Windows.
